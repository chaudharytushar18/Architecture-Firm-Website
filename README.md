# Aakaar Studio

A single-page architecture studio website built as a **portfolio demonstration project** with fictional Indian studio details. All names, addresses, phone numbers, emails, and project information are placeholder content.

## Live preview

Open `index.html` in a browser, or serve the folder locally:

```bash
npx serve .
```

Then visit `http://localhost:3000`.

## Project structure

```
├── index.html          Entry page
├── README.md
├── css/
│   └── styles.css      Site styles
├── js/
│   └── main.js         Navigation, portfolio, modal, form demo
└── assets/
    └── images/
        ├── hero-bg.jpg
        └── hero-bg.jpg
```

## Features

- Responsive navigation with **Home**, About, Services, Portfolio, Careers, and Contact
- Hero section with GSAP entrance animation
- Portfolio categories with project grid and detail modal
- Demo contact form (shows confirmation only; no backend)
- Awards and testimonials sections on the home view

## Tech stack

- HTML5, CSS3, vanilla JavaScript
- [GSAP](https://greensock.com/gsap/) for hero animations
- [Font Awesome](https://fontawesome.com/) icons
- Google Fonts (Roboto, Poppins)

## Notes for portfolio use

- Replace placeholder copy and images with your own work when showcasing the template.
- Team photos use Unsplash URLs; swap for local assets if you prefer offline hosting.
- Social links point to `example.com` — update before publishing.
